Windows 11 上安装 Git 的步骤：

  1. 打开浏览器，前往 Git 官方网站：https://git-scm.com/downloads

  2. 在页面中选择 “Windows” 操作系统，并下载最新的安装程序。你可以根据你的系统架构选择 32 位或 64 位版本。

  3. 下载完成后，双击运行安装程序。在弹出的安装窗口中点击“下一步”。

  4. 阅读许可协议并同意，然后再次点击“下一步”。

  5. 选择安装路径或保留默认设置，然后点击“下一步”。

  6. 选择组件，建议选择默认选项，然后点击“下一步”。

  7. 配置安装选项，建议选择默认设置，并确保勾选 “Enable Git Credential Manager” 和 “Use OpenSSL library” 选项，然后点击“下一步”。

  8. 选择启动菜单文件夹，建议保留默认设置，然后点击“下一步”。

  9. 选择额外任务，建议保留默认设置，然后点击“下一步”。

  10. 点击“安装”按钮开始安装 Git。

  11. 安装完成后，在“完成”窗口中，确保选中 “Launch Git Bash” 选项，然后点击“完成”。

Windows 11 上安装 Java 64 位：

  1. 打开浏览器，前往 Java 的官方网站 (https://www.java.com/en/download/manual.jsp)

  2. 点击“Download”按钮以下载 Java 安装程序。

  3. 在下载对话框中选择“保存文件”。

  4. 下载完成后，双击打开安装程序文件。

  5. 在弹出的安装向导窗口中，选择“安装”选项。

  6. 稍等片刻，直到安装程序将 Java 安装至您的计算机。

  7. 在安装完成后，关闭安装程序窗口。

  8. 检查是否成功安装 Java。打开命令提示符或 PowerShell 终端，输入“java -version”并按回车键。如果成功安装了 Java，则会显示已安装的 Java 版本信息。

恭喜你，已经成功在 Windows 11 计算机上安装了 Java 64 位。

Windows 11上安装NVIDIA Broadcast：

  1. 首先，确保您的计算机安装了最新版的NVIDIA显卡驱动程序。如果您不确定，请前往 NVIDIA官方网站 下载并安装最新版本的显卡驱动。

  2. 前往NVIDIA Broadcast官网https://www.nvidia.cn/geforce/broadcasting/broadcast-app/ 下载NVIDIA Broadcast应用程序。

  3. 安装NVIDIA Broadcast应用程序。双击下载好的“NvBroadcastInstaller.exe”文件，按照提示完成安装过程。

  4. 启动NVIDIA Broadcast应用程序。您可以在Windows 11的开始菜单中找到该应用程序图标，单击以启动它。

  5. 在NVIDIA Broadcast应用程序中，您将看到可以配置和使用的三个功能：声音，相机和广播。根据您的需求配置每个功能即可，例如选择默认麦克风，摄像头等。

  6. 当您完成所有设置后，单击“启动广播”按钮即可开始使用NVIDIA Broadcast进行直播或录制视频。
